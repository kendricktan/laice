  Build your first app.


Here you'll learn how to get your first NLP app up and going in the cloud

1. Create your first story

In order for Laice to classify the text you sent it, it needs some context. The context needed is stored in a Story. Stories can be made here.

2. Establishing attributes

Attributes are like features that Laice tries to identify within a sentence. For example, if you have two attributes: name, and location and you send it the sentence: John is in London, Laice will try and associate {'name': 'John', 'location': 'London'} Of course this doesn't happen by magic and will require some training before Laice is able to do that, which is what we'll look into next!

3. Added some examples

Laice isn't powered by black magic and will of course need some examples before it'll be able to automatically recognize attributes on its own. Start by entering some basic query examples for Laice to learn.

4. Configuring your examples

Once you have a satisfiable amount of examples (5-10), start configuring where each attributes should belong to in a sentence. You can configure which attributes belong to which part of the sentence via the dropdown box.

5. Train it!

Now that we have a sufficient list of examples to follow, simply click on the Retrain model button located at the top of your project.

6. Test it!

Now simply go back to the manual query part of your story and insert a new sentence. Laice hopefully *fingers crossed* should be able to correctly label the key attributes of the new sentence. If it doesn't work then continue reading below.

6. Passing data through it!

You can now communicate with the server via a RESTful request. To do it with Curl execute: 

curl -X post -H "Content-Type: application/json" -d '{"querystring": "Jack is in London"}' http://127.0.0.1:8000/api/stories/my-story-name/queries/
Note: You might need to change 127.0.0.1:8000 to your server domain and my-story-name to your story's name. The curl request is also located at the very top of each of your story.

Help! My model isn't working!

Laice learns by examples, if it isn't able to classify your sentence correctly just reconfigure the sentence it got wrong via the dropdown buttons/delete buttons and hit the retrain button
